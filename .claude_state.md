# Claude Project State — speca-piano-coach
_Last updated: 2026-05-31 (Session 3)_

## Stack
- **Frontend**: Vanilla HTML5 / CSS3 / JavaScript ES6 (IIFE modules)
- **Audio**: Web Audio API (hybrid: sample engine + oscillator fallback)
- **Visualization**: HTML Canvas API
- **Storage**: localStorage
- **Testing**: Vitest (74 tests ready, needs Node.js)
- **No framework, no bundler** — opens directly in browser

---

## Kiến trúc hiện tại (SPA với Router)

```
App Shell (top-bar + bottom-nav)
├── View: home        → HomeView.js     (lesson path, CTA, stats)
├── View: learn       → LearnView.js    (full-screen lessons)
├── View: practice    → inline HTML     (piano + chords + songs)
└── View: profile     → ProfileView.js  (XP, badges, streak)
```

Navigation: `Router.go(viewId)` → `js/router.js`
Bootstrap:  `AppShell.boot()` → `js/appShell.js`
