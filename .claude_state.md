## Stack
- **Frontend**: Vanilla HTML5 / CSS3 / JavaScript ES6 (IIFE modules)
- **Audio**: Web Audio API — hybrid SampleEngine + oscillator fallback
- **Visualization**: HTML Canvas (Visualizer + FallingNotes)
- **Storage**: localStorage (progress, preferences)
- **PWA**: manifest.json + service worker (offline)
- **Testing**: Vitest, 74+ tests (needs Node.js)
