# Claude Project State — speca-piano-coach
_Last updated: 2026-05-31_

## Stack
- **Frontend**: Vanilla HTML5 / CSS3 / JavaScript ES6 (IIFE modules)
- **Audio**: Web Audio API (oscillator synthesis)
- **Visualization**: HTML Canvas API
- **Storage**: localStorage
- **Testing**: Vitest (ready; requires Node.js to run)
- **No framework, no bundler, no npm dependencies** at runtime
